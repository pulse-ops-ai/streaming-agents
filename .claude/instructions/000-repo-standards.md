# 000 – Repo Standards

- **Monorepo**: pnpm workspaces + Terraform modules
- **Language split**: TypeScript (NestJS) for services, Python (uv) for ML/LLM agents
- **Formatting**: Prettier (TS), Ruff (Python)
- **Commits**: Conventional Commits (`feat:`, `fix:`, `chore:`, …)
- **Branch strategy**: trunk-based (short-lived feature branches)
